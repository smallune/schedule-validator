How It Works
  1. Import — Upload the Room Grid and Enrollment Excel files (or use Quick Import if they're already in the project folder).
  2. Add Constraints — Enter any professor availability notes, room restrictions, or scheduling preferences under Constraints.
  3. Generate — Click Generate Schedule with AI. Claude reviews all data, checks for conflicts, and proposes assignments.
  4. Review — Go to the Schedule grid. Confirm entries that look good, reject ones that don't, or move them to a different room / time.
  5. Iterate — Add new constraints as information arrives and re-generate. Confirmed entries stay locked.
